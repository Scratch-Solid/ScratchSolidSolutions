# ScratchSolidSolutions Monorepo

This repository contains:
- `frontend`: Modern React app (slim, secure template)
- `backend`: Python FastAPI app (secure, world-class API)

## Security
- Security is enforced at every layer: sanitization, secure headers, authentication, and more.
- CI/CD pipeline will include security checks and automated deployment to Cloudflare.

## Structure
- `frontend/` — React app
- `backend/` — FastAPI app
- `.github/` — CI/CD and automation

Instructions for local development, security, and deployment will be added as setup progresses.
